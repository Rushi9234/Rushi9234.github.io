Transformer-based computer vision + NLP pipeline that generates natural-language captions for any image. Vision Transformer (ViT) encoder extracts deep image features; GPT-2 decoder generates fluent captions. Built using Hugging Face VisionEncoderDecoderModel — an advanced architecture over conventional CNN+RNN approaches. Deployed end-to-end with a Gradio interface for real-time image upload and instant caption output.

Architecture: ViT encoder + GPT-2 decoder · Framework: Hugging Face Transformers · Deployed: Gradio on Colab
